## How Font Selection Works

1. Drop `.ttf` or `.otf` files into this directory
2. Open the GUI → **Generate Video** → **Font** dropdown
3. Select your font — preview updates live
4. Generate! The selected font is used for all text overlays

Supported formats: `.ttf`, `.otf`, `.woff`, `.woff2`
